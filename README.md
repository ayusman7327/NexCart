# 🛒 NexCart AI

> An AI-powered full-stack eCommerce platform built with the MERN Stack, featuring secure authentication, modern UI, and intelligent shopping experiences.

---

## 🚀 Overview

NexCart AI is a modern full-stack eCommerce application designed to deliver a seamless online shopping experience. The platform enables users to browse products, manage their shopping cart, securely authenticate, and place orders through a responsive and intuitive interface.

The project follows a scalable MERN architecture and is built with clean, modular code suitable for real-world applications.

## Install Frontend

```bash
cd frontend
npm install
```

---

## Install Backend

```bash
cd ../backend
npm install
```

---

# 🔑 Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```

---

# ▶️ Run Backend

